    *   *Overview:* Standardized, deduped, and cleaned customer datasets containing ~5,000 rows.
    *   *Key Insight:* Conducted deep aggregations using JOIN and COUNT queries to uncover purchase and order-value patterns.
3.  **👥 HR Recruitment Analytics (Power BI & Excel)**

## 💻 Tech Stack & Features of the Portfolio Website

This portfolio website was designed from scratch to serve as a highly professional, fast-loading, and interactive showcase.

*   **Structure:** Semantic HTML5
*   **Styling:** Custom Vanilla CSS (featuring a modern Bento Grid system, CSS variables, and fluid responsive layouts)
*   **Interactivity:** Vanilla JavaScript
    *   **Dynamic Theme Toggle:** Smooth transition between custom dark-mode and light-mode color palettes.
    *   **Interactive Filters:** Smooth, category-based project filtering (All, Power BI, SQL, Python) without page reloads.
    *   **Project Details Modal:** A slide-up popup modal presenting metrics and source links for each selected project.
*   **Aesthetics:** Premium typography (Outfit/Inter-style design), micro-animations, bento styling, and abstract vector grid decorations.
